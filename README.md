# CustodiApp

CustodiApp es una aplicación diseñada para visualizar calendarios de custodia compartida de forma rápida y clara.

La app permite generar calendarios anuales de semanas alternas y otros patrones habituales en convenios reguladores sin necesidad de cálculos manuales.

Características:

- Calendario visual por colores
- Configuración de semanas alternas
- Vacaciones y excepciones
- Uso completamente local en el dispositivo
- Sin cuentas ni sincronización en la nube
